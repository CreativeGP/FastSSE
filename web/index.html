What is FastSSE?

FastSSE is a simple library to create a realtime web app with the SSE (Server-Sent Events). It's composed of JavaScript and PHP code. You can create a web app easily and fast by using them.

Here's the website of FastSSE as you know. You can find the informations about FastSSE:

What is SSE?

SSE is a technology to use the server->client connection over HTTP. It's very convenient and can run even in shared server.

This is the Improved Comet (Comet).

In Comet, we have to send a request to watch the event from server. This way may cause a lag.

To improve this problem, SSE leverages the chunked response in HTTP. So the client can watch the event in only none connection.

Installation

  1. Move the `client` folder in the directory that the client-side program has been put.
  2. Move the `fastsse` folder in the directory that the server-side program has been put.
  3. Include library files in the respective way, client-side and server-side (i.e. JavaScript and PHP for now).

Downloads

Please clone the GitHub repository or download library files.
