Architecture

The left picture shows the architecture of FastSSE. This is little complicated as you see. Let's break it down.

Types of component

There are some types of component:

  • Client Interfaces
  • Server Interfaces (SI)
  • Buffers

The color of each type corresponds the picture.

First, FastSSE divides the world into three regions to provide realtime connections even in shared severs:

Client-Side, Server Interface, Server Storage

Each region is separated by the black line in the picture. The left side is Client, the upper-right space is Server Interface, and the lower-right space is Server Storage.

Client-Side

This is a client, in which runs JavaScript. You can see two red rectangles named "OnMessage" and "OnError", these are event handlers (Client Interfaces).
"OnMessage" is called when the client received data, "OnError" is called when errors occured.
This is very simple.
